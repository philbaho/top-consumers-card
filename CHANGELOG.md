# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added
- Initial release
- Auto-sorting by power consumption
- 28 available icons (emojis)
- 12 color gradients
- Support for 1-20 items display
- Responsive design for desktop and mobile
- Click to view entity details
- Real-time power monitoring
- Modern gradient design with animations

### Features
- Display top N power consumers
- Configurable title and max items
- Customizable icon and gradient per entity
- Automatic unit detection (W, kW, etc.)
- Smooth hover animations
- Dark mode support
